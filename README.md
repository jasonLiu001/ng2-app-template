ng2-app-template
===================
a cli tool for ng2 to create module.
-------------------

## Install
    ```
        npm install -g ng2-app-template
    ```
    
## Usage
    ```
        ng2-app-template --help
    ```
    
## CreateModule
    ```
        ng2template --module-name <module-name>  //create a module named with <module-name>
    ```
